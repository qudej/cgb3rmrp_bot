📋 Основной функционал бота:
1. 💼 Кадровый аудит и заявки
Трудоустройство и Гос. сотрудники: Пользователи подают заявки через модальные окна (с валидацией статического ID в формате 123-456).
Увольнение и ЧС: Бот автоматически снимает все фракционные роли, выдает роль «Гражданин» и меняет никнейм на формат УВ | Имя Фамилия | 123-123.
Логирование: Все действия сопровождаются эмбедами в лог-канале с прямыми ссылками на оригинальные заявки.


2. 🏥 Управление отделами (БСМП, АБ, КУЦ)
При одобрении перевода бот забирает роль старого отдела и выдает новую.
Меняет тег отдела в никнейме (например, БСМП | Имя Фамилия | 123-456).
При подаче заявки в отдел бот тегает Заведующих и Замов конкретного отдела.


3. 📈 Система рангов
Выдача и снятие ролей согласно иерархии в config.py.
Исходя из отдела в котором состоит человек, и при достижении определенного ранга выдает ему  роль (например, «Старший БСМП»).
При повышении на определенные должности (например, Санитар), бот сам подставит нужный префикс в никнейм (например, ВБО | ...).


5. 📦 Логистика и поставки
Создание заявок на поставки (ЗМХ / МС) с указанием Фракции и Срочности.
Встроенные кнопки для Старшего Состава: Успешно, Частично успешно (с указанием деталей), Выбили, Отказано (с указанием причины).
Формирование и отправка итогового отчета в канал логистики.


7. 🔨 Дисциплинарные взыскания
Выдача выговоров и предупреждений через выпадающее меню с указанием причины.
Публикация отчета о взыскании.


🎮 Инструкция использования:

1. Вызов интерфейсов (Панелей)
Напишите команду !setup в текстовом канале запрос роли. Бот создаст 4 интерактивных панели (Кадры, Взыскания, Поставки, Отделы).

2. Контекстное меню
Для быстрых действий с сотрудниками:

Кликните правой кнопкой мыши по сотруднику -> Приложения (Apps) -> Выберите нужное действие:

+1 ранг — Быстро повысить сотрудника на 1 ступень.

Установить ранг — Откроет меню для выбора конкретного ранга.

Уволить — Откроет форму для быстрого увольнения с причиной и ЧС.

Выдать взыскание — Откроет меню для выдачи выговора.
Внимание: Все кнопки и контекстные меню защищены. Если обычный игрок попытается их нажать, бот выдаст ошибку отсутствия доступа.


🛠 Техническая часть и настройка


Структура проекта:
main.py — Точка входа, запуск бота и регистрация контекстных меню.
config.py — Главный файл настроек. Здесь находятся все ID ролей, каналов, списки пингов и сетка рангов.
utils.py — Системные функции (проверки прав, парсинг никнеймов, логика увольнения).
views_hr.py — Логика кадров.
views_dept.py — Логика отделов.
views_punish.py — Логика выговоров.
views_supply.py — Логика поставок.
views_ranks.py — Логика выдачи рангов.
