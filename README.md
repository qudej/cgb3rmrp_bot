📋 Основной функционал бота
1. 💼 Кадровый аудит и заявки
Трудоустройство и Гос. сотрудники: Пользователи подают заявки через удобные модальные окна (с жесткой валидацией статического ID в формате 123-456).
Увольнение и ЧС: Бот автоматически снимает все фракционные роли, выдает роль «Гражданин» и меняет никнейм на формат УВ | Имя Фамилия | 123-123.
Логирование: Все действия сопровождаются красивыми эмбедами в лог-канале с прямыми кликабельными ссылками на оригинальные заявки.

2. 🏥 Управление отделами (БСМП, АБ, КУЦ)
Умная выдача ролей: При одобрении перевода бот сам забирает роль старого отдела и выдает новую.
Смена никнеймов: Автоматически меняет тег отдела в никнейме (например, БСМП | Имя Фамилия | 123-456).
Упоминания (Пинги): При подаче заявки в отдел бот автоматически тегает Заведующих и Замов конкретного отдела.

3. 📈 Система рангов
Автоматизация повышений: Полностью автоматическая выдача и снятие ролей согласно иерархии в config.py.
Роли Старших сотрудников: Бот понимает, в каком отделе состоит человек, и при достижении 5+ ранга выдает ему правильную роль (например, «Старший БСМП»). При смене отдела на 5+ ранге эта роль автоматически заменяется.
Умные префиксы: При повышении на определенные должности (например, Санитар), бот сам подставит нужный префикс в никнейм (например, ВБО | ...).

4. 📦 Логистика и поставки
Создание заявок на поставки медикаментов (ЗМХ / МС) с указанием Фракции заказчика и Срочности.
Встроенные кнопки для Старшего Состава: Успешно, Частично успешно (с указанием деталей), Выбили, Отказано (с указанием причины).
Автоматическое формирование и отправка итогового отчета в канал логистики.

5. 🔨 Дисциплинарные взыскания
Выдача выговоров и предупреждений через удобное выпадающее меню с указанием причины (пункта устава).
Автоматическая публикация отчета о взыскании.


🎮 Как пользоваться старшему составу?
Управление ботом максимально упрощено и не требует запоминания команд.

1. Вызов интерфейсов (Панелей)
Напишите команду !setup в любом текстовом канале (требуются права Администратора). Бот создаст 4 интерактивных панели (Кадры, Взыскания, Поставки, Отделы). Эти панели вечные — они продолжат работать даже после перезапуска бота.

2. Контекстное меню (ПКМ)
Для быстрых действий с сотрудниками используйте контекстное меню Discord:

Кликните правой кнопкой мыши по сотруднику -> Приложения (Apps) -> Выберите нужное действие:

+1 ранг — Быстро повысить сотрудника на 1 ступень.

Установить ранг — Откроет меню для выбора конкретного ранга.

Уволить — Откроет форму для быстрого увольнения с причиной и ЧС.

Выдать взыскание — Откроет меню для выдачи выговора.
Внимание: Все кнопки и контекстные меню надежно защищены. Если обычный игрок попытается их нажать, бот выдаст ошибку отсутствия доступа.

🛠 Техническая часть и настройка (Для Разработчиков)

Бот построен на модульной архитектуре (Cogs/Views), что делает его быстрым и невероятно легким в поддержке.

Структура проекта:
main.py — Точка входа, запуск бота и регистрация контекстных меню.
config.py — Главный файл настроек. Здесь находятся все ID ролей, каналов, списки пингов и сетка рангов. Если на сервере удалили роль или канал — просто поменяйте ID в этом файле.
utils.py — Системные функции (проверки прав, парсинг никнеймов, логика увольнения).
ui/ — Папка с пользовательскими интерфейсами (модальные окна и кнопки):
views_hr.py — Логика кадров.
views_dept.py — Логика отделов.
views_punish.py — Логика выговоров.
views_supply.py — Логика поставок.
views_ranks.py — Логика выдачи рангов.

Запуск бота:
Установите зависимости: pip install discord.py python-dotenv

Создайте файл .env в корневой папке проекта и добавьте туда токен бота:

DISCORD_BOT_TOKEN=Ваш_Секретный_Токен

Запустите скрипт: python main.py
